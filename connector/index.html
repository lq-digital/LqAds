Overview

The LQ Analytics connector allows you to securely connect your Facebook Ads data to Looker Studio. It enables real-time reporting by retrieving performance metrics such as clicks, impressions, conversions, and ad spend directly from Facebook's Graph API.

Key Features

  • Real-time Facebook Ads data integration
  • Custom dashboard creation in Looker Studio
  • Secure OAuth authentication via Google and Facebook
  • Read-only access (no data modification)

How It Works

  1. Open Looker Studio
  2. Click "Add Data Source"
  3. Select "Facebook Ads (LQ Analytics)"
  4. Authorize with your Google account
  5. Log in to your Facebook account
  6. Select your ad account
  7. Click "Connect" to start using your data

Requirements

  • A valid Google account
  • Access to Facebook Ads (Business Manager or Ad Account)
  • Permission to view ad performance data

Data Access & Permissions

The connector uses the following permissions:

Google Scopes

  • userinfo.email – Identify the signed-in user
  • script.storage – Store connector configuration
  • script.external_request – Fetch data from Facebook API

Facebook Access

  • Read-only access to Facebook Ads data
  • No posting, editing, or deletion permissions

Security & Privacy

  • OAuth authentication ensures secure login
  • No passwords are stored
  • No data is modified or deleted
  • Data is retrieved on request from Facebook APIs
  • All operations are read-only and transparent

Data Usage

The connector only retrieves the data required to generate reports in Looker Studio. No personal data is sold or shared with third parties.
